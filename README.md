Welcome to the Ammo Managment System or AMS for short!

This is a mod designed to overhaul Terraria's ranger class by making the selection, storage and switching of ammo intuitive and interactive.

I am currently learning how to code in C# so please be patient as it might take a while for me to get this from a basic mod to a fully fleshed out version!

If you have any amazing ideas for me please drop them in the comments as I would love some community engagement and feedback!
